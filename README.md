- ⚡️ [TypeScript](https://www.typescriptlang.org/)

- ⚡️ [PNPM](https://pnpm.io/motivation)

## Installation

```bash

# install dependencies
- git clone https://github.com/bayudha2/placeholder-app.git
- cd placeholder-app
- npm install -g pnpm
- pnpm install

# start the application
pnpm run start

# make a production build
pnpm run build
```
